# Sistema de Procesamiento de Pagos

## Descripción

Aplicación de consola desarrollada en C# para gestionar distintos métodos de pago.

El sistema permite:

- Registrar pagos.
- Mostrar información de los pagos.
- Procesar operaciones.
- Calcular comisiones según el método de pago.
- Mostrar el monto final de la operación.
